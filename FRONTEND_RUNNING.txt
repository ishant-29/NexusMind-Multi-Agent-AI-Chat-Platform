• Chat won't work without Agent Service
• Authentication won't work without Auth Service
• Document upload won't work without File Service
• Conversations won't save without Conversation Service

═══════════════════════════════════════════════════════════════════════════════

🚀 START ALL SERVICES
═══════════════════════════════════════════════════════════════════════════════

To start all backend services, run:
  start-services.bat

Or start them individually:

Terminal 1 - Agent Service:
  cd services/agent-service
  python main.py

Terminal 2 - Auth Service:
  cd services/auth-service
  npm run dev

Terminal 3 - Conversation Service:
  cd services/conversation-service
  npm run dev

Terminal 4 - File Service:
  cd services/file-service
  npm run dev

Terminal 5 - User Service:
  cd services/user-service
  npm run dev
