❌ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: Service won't start
Solution:
  • Check if port is already in use
  • Verify .env files are configured
  • Check MongoDB connection string
  • Review terminal window for errors

Problem: "Cannot connect to MongoDB"
Solution:
  • Verify MONGODB_URI in .env files
  • Check MongoDB Atlas is accessible
  • Ensure IP is whitelisted in Atlas
  • Check internet connection

Problem: "API key not found"
Solution:
  • Verify API keys in .env files:
    - OPENAI_API_KEY (for embeddings)
    - GROQ_API_KEY (for LLM)
    - TAVILY_API_KEY (for web search)
  • Restart the service after adding keys

Problem: Frontend shows "Service unavailable"
Solution:
  • Wait for all services to fully start
  • Check service health endpoints
  • Verify service URLs in .env.local
  • Check terminal windows for errors

Problem: Port already in use
Solution:
  • Run: kill-services.bat
  • Wait 10 seconds
  • Run: start-services.bat again
