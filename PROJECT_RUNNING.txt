┌──────────────────────────────────────────────────────────────────────────────┐
│                         TROUBLESHOOTING                                      │
└──────────────────────────────────────────────────────────────────────────────┘

If you see errors:

❌ "Port already in use"
   → Run: kill-services.bat
   → Wait 10 seconds
   → Run: start-services.bat

❌ "Cannot connect to MongoDB"
   → Check MONGODB_URI in .env files
   → Verify MongoDB Atlas is accessible
   → Check IP whitelist in Atlas

❌ "API key not found"
   → Check .env files have all API keys:
     • OPENAI_API_KEY
     • GROQ_API_KEY
     • TAVILY_API_KEY
   → Restart services after adding keys

❌ "Module not found"
   → Run: npm install (in frontend)
   → Run: npm install (in each service)
   → Run: pip install -r requirements.txt (in agent-service)
